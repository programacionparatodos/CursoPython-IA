# 🐍 Curso de Python + Inteligencia Artificial

¡Bienvenidos al repositorio oficial del curso! Aquí encontrarás todo el material didáctico, códigos de ejemplo y recursos complementarios que utilizaremos a lo largo de las clases.

## 📂 Contenido del Repositorio
* **Códigos fuente:** Ejemplos prácticos explicados paso a paso.
* **Presentaciones:** Diapositivas de soporte teórico para las clases.
* **Recursos:** Enlaces, conjuntos de datos (datasets) y lecturas recomendadas.

## 🛠️ Requisitos previos
Para ejecutar los códigos de este curso, asegúrate de tener instalado:
* Python 3.10 o superior
* Un editor de código (como VS Code) o entornos como Jupyter Notebook / Google Colab

## 🚀 Cómo usar este material
1. Clona este repositorio en tu computadora.
2. Explora las carpetas organizadas por módulos o clases.
3. Ejecuta los scripts de Python para poner en práctica lo aprendido.
